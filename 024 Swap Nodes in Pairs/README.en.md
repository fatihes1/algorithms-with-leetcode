#### Example 1:

Input: head = [1,2,3,4] <br>
Output: [2,1,4,3]<br>

#### Example  3:

Input: head = [1]<br>

### Constraints:

The number of nodes in the list is in the range [0, 100].<br>
0 <= Node.val <= 100<br>
